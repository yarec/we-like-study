weLikeStudy V0.9(Beta)

这个插件提供一个“网络学习平台”,目前功能有:

1 在线做习题
 1.1 做单一的一道习题
 1.2 做同一种题型的多道题
 1.3 做一张试卷
  1.3.1 题目导航
  1.3.2 时间记录
  1.3.3 记录每一次做试卷的过程
2 统计自己历次考试的成绩变化
3 查看自己的错题记录
 3.1 复习自己的错题本
4 查看自己的历次考试记录
5 集成到Discuz

计划升级的功能有:

1 能够集成到各种主流的CMS系统中,包括Discuz,phpWind,Joomla,Wordpress,Drupal,PhpBB,织梦,帝国
 1.1 能直接以.RAR包上传到CMS插件安装口后集成
 1.2 能够支持MYSQL 和 PostGreSql 两种数据库
 1.3 首先做好Discuz的集成
2 提供一个试卷同步接口
 2.1 被同步的网站开放一些权限,能够让其他网站直接在线同步试卷数据,同步格式为XML
 2.2 能够导入导出EXCEL文件
3 做一个在线考试功能
 3.1 在线考试功能的参加者有 学员和监考老师
 3.2 每个参加者的电脑都要配一个摄像头,系统可以每隔一段时间拍照并将参考者的考试现状发到服务器
 3.3 监考老师可以实时监督任何一个考生的考试现状
4 做一个基于课程的在线学习模块
 4.1 采用类似PPT的教学过程(参考 我要学外语 这个网站的设计)
5 让系统能够运行在多个平台上
 5.1 前台采用 EXT ,FLEX , 银光 , DHTML5 
 5.2 后台采用 .NET(C#) ,JAVA , Python
